# CrazySpider

## 项目宗旨

持续收集具有信息差的优质产品和有效推广渠道，并持续沉淀、提炼这些信息的发现方法与验证方法，帮助我们在市场中更快识别可复制的增长机会。

## 我们关注什么

- 有信息差的好产品：在细分市场中更早看到尚未被大规模模仿、并已展现增长潜力的产品。
- 优秀的推广渠道：可持续带来高质量流量与转化、且具有复用价值的渠道。
- 信息收集方法论：不仅“收集了什么”，更关注“如何找、如何验证、如何复用”。

## 目标

1. 建立稳定的情报输入机制，减少盲目试错。
2. 通过结构化方式复盘每次发现，形成可复用的筛选标准。
3. 最终输出可执行的增长候选清单，并推动快速验证与迭代。
